# Run Local (5 Steps)

1. Open project folder in VSCode.
2. Install **Live Server** extension.
3. Right-click `public/index.html` 창혛혪 **Open with Live Server**.
4. Use URL like: `http://127.0.0.1:5500/public/index.html`.
5. Test flow: Login 창혛혪 Add to cart 창혛혪 Proceed to Checkout 창혛혪 Place order.

## Quick Fixes
- If page is stale: press `Ctrl + F5`.
- If checkout fails: confirm `public/checkout.html` exists.
- Do not open with `file://` URL.
